# GenLayer 测试网区块链浏览器

## 项目概述
这是一个为 GenLayer 测试网构建的全面区块链浏览器，包含交易历史记录、区块详情、验证者信息和实时网络统计数据。

## 开发信息
- **完成时间**: 2025年12月19日


## 功能特性

### 1. 仪表板
- 实时网络统计数据
- 总区块数、总交易数、活跃验证者数量
- 网络算力显示
- 最新区块和交易预览

### 2. 区块浏览器
- 完整的区块列表展示
- 区块详细信息查看
- 区块搜索功能
- 分页浏览支持

### 3. 交易浏览器
- 交易历史记录查询
- 交易详情展示
- 交易状态跟踪
- 交易哈希搜索

### 4. 验证者信息
- 验证者列表展示
- 验证者状态监控
- 质押信息显示
- 正常运行时间统计

## 技术架构

### 后端技术
- **Node.js**: 服务器运行环境
- **Express.js**: Web框架
- **Socket.io**: 实时数据推送
- **CORS**: 跨域资源共享

### 前端技术
- **HTML5**: 页面结构
- **CSS3**: 样式设计（响应式布局）
- **JavaScript**: 交互逻辑
- **Socket.io Client**: 实时数据接收

### 数据模拟
- 模拟区块链数据生成
- 实时数据更新机制
- RESTful API 接口设计

## 安装和运行

### 方法一：使用启动脚本（推荐）
1. 双击运行 `start.bat` 文件
2. 等待依赖安装完成
3. 服务器将自动启动

### 方法二：手动启动
1. 安装依赖：
   ```bash
   npm install
   ```

2. 启动服务器：
   ```bash
   npm start
   ```

3. 打开浏览器访问：
   ```
   http://localhost:3000
   ```

## API 接口

### 区块相关
- `GET /api/blocks` - 获取区块列表
- `GET /api/blocks/:number` - 获取指定区块详情

### 交易相关
- `GET /api/transactions` - 获取交易列表
- `GET /api/transactions/:hash` - 获取指定交易详情

### 验证者相关
- `GET /api/validators` - 获取验证者列表

### 统计数据
- `GET /api/stats` - 获取网络统计数据

## 实时功能
- WebSocket 连接实现实时数据推送
- 网络统计数据每5秒更新一次
- 自动刷新最新区块和交易信息

## 响应式设计
- 支持桌面端和移动端访问
- 自适应布局设计
- 优化的用户体验
