# GenLayer 测试网区块链浏览器

一个全面的区块链浏览器，包含交易历史记录、区块详情、验证者信息和实时网络统计数据。

## 功能特性
- 交易历史记录查询
- 区块详情展示
- 验证者信息管理
- 实时网络统计数据
- 响应式设计

## 技术栈
- React.js
- Node.js
- Express.js
- Web3.js
- CSS3
